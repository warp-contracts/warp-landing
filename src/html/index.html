Blockchain data problem

Most smart contracts treat data as a side-product that slows down processing. Data is digested, pruned and pushed out to side chains. Nonetheless the bloated global state is still the biggest challenge of modern blockchains.

How SmartWeave solves that

We use a radically different design treating data as the most valuable asset. We get rid of a global state thanks to decoupling the storage from the computation. The contracts can be evaluated on-demand, lazily, creating the potential for unparalleled scalability.
